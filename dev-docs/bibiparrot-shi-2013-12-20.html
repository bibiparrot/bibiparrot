Development Environment^

Python 2.7+

Note: There is a known compatibile issue about python 2.7.2. As when try to import the wxWidgets, the "Dll load error" will occur, and the program will abort.
To solve this issue, we suggested to use the python version above 2.7.5+ and below 3.0-. It seems that python version 2.7.2 has compatible issues loading wxWidgets graphic dlls.

wxPython 2.8.8+ (32bit)

The wxPython demo package of 2.8 is also very important.

The problem is that WxPython is only available on the Mac in 32-bit mode; however, by default, Python will start up in 64-bit mode. To fix this problem, create the following shell script named python_32:

# WxPython Incompatible With Snow Leopard? #! /bin/bash export VERSIONER_PYTHON_PREFER_32_BIT=yes # defaults write com.apple.versioner.python Prefer-32-Bit -bool no # If you decide you want to switch back into 64-bit mode, you can then use the following command: defaults write com.apple.versioner.python Prefer-32-Bit -bool yes

since you might not have enough shell scripting knowledge (you need to use "./python_32" or place it in a folder that is listed in your "$PATH" environment variable and invoke it as "python_32") to follow the former option, you might want to simply execute the following command which will make 32-bit mode the default:

pycharm Free Community Edition (suggested editor)

Of course, any editor will be fine. Only pyCharm is perfect for function reference and friendly for newbies (that's us).

Third Party Installation

  • The audio play depends on pygame 1.9.1+.
  • The export to MS Word 2007 (*.docx) depends on docx 0.2.3 (source ) which depends on lxml.
  • The speech recognition will use the Google Speech API, (e.g. PyGsr) which depends on pyAudio for recording the local audio. While converting from local .wav file into .flac file needed flac library, because .flac is needed for Google speech API.

Most of these library can be find installers. (e.g. Windows Installer).

UI Design^

UI Elements Types

  • MainFrame: main frame organizes the elements components
  • MainMenu: the menu for the main frame
  • MainToolbar: the toolbar for the main frame
  • MainTabs: the overview explorer
  • MainStatusbar: the status bar for the main frame
  • RecordsList: the list of the records (mp3).
  • RepeaterControl: The interface control of play a record.
  • Repeater: the frame (with toolbar) of the record controller
  • RepeatList: the list of the repeat breaks
  • EditControl: the interface control of editor
  • Editor: the frame of editing (with toolbar)

Each element has full screen version (resize, dynamically display, repaint)

Python to Installation^

We need py2exe or py2app for compiling installation module.

Py2exe may have problem to load 3rd party library, thus we need add parameters to tell the module. e.g.

# e.g. lxml is required. $ python setup.py py2exe -p win32com,lxml -i twisted.web.resource ↵
