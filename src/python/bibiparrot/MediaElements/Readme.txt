Readme File for audioUtils
====================================

Installation Instructions for the audio library:

PyGame Library

Windows 32 Platform
----------------------
In the audioLibs folder, double click the pygame installer package.
Just follow the instructions, and the library could be installed by using "import pygame" command.

Mac OSX Platform
----------------------
Just follow the instructions of the apple install package, and you can import the library using "import pygame" command.

** Note: Windows 64 platform might not be compatibile with the pygame library.

** Note: The test musics are in testMusics for testing.
